# Guide Utilisateur - Proprietaire

## Bienvenue sur Gestoo

Gestoo est la plateforme nationale de gestion des hebergements touristiques au Senegal. Ce guide vous aidera a utiliser la plateforme pour gerer vos proprietes, enregistrer vos clients et payer la Taxe de Promotion Touristique (TPT).

## Inscription

### Via le portail web

1. Rendez-vous sur **app.gestoo.sn**
2. Cliquez sur **"S'inscrire"**
3. Entrez votre numero de telephone (format: +221 7X XXX XX XX)
4. Vous recevrez un code de verification par SMS
5. Entrez le code pour valider votre numero
6. Remplissez votre profil:
   - Nom complet
   - Numero de CNI
   - Photo de votre CNI (recto)
   - Nom de l'entreprise (optionnel)
   - Numero NINEA (optionnel)
7. Cliquez sur **"Creer mon compte"**

### Via WhatsApp

1. Envoyez **"Bonjour"** au numero **+221 XX XXX XX XX**
2. Selectionnez **"Nouveau proprietaire"**
3. Suivez les instructions:
   - Entrez votre nom complet
   - Entrez votre numero de CNI
   - Envoyez une photo de votre CNI
4. Confirmez vos informations

## Enregistrer une propriete

### Via le portail web

1. Connectez-vous a votre compte
2. Dans le menu, cliquez sur **"Proprietes"**
3. Cliquez sur **"Ajouter une propriete"**
4. Remplissez le formulaire:

| Champ | Description |
|-------|-------------|
| Nom | Nom de votre etablissement |
| Type | Hotel, Meuble, Maison d'hotes, Location courte duree |
| Adresse | Adresse complete |
| Ville | Ville (ex: Dakar, Saly, Saint-Louis) |
| Region | Region administrative |
| Capacite | Nombre de chambres/lits |

5. Ajoutez des photos de votre propriete (minimum 1)
6. Partagez votre position GPS (optionnel mais recommande)
7. Cliquez sur **"Soumettre"**

Votre demande sera examinee sous 48 heures. Une fois approuvee, vous recevrez votre **numero de licence Gestoo** (format: GST-2024-XXXXX).

### Via WhatsApp

1. Tapez **"menu"** puis selectionnez **"Ajouter propriete"**
2. Entrez le nom de votre etablissement
3. Choisissez le type dans la liste
4. Entrez l'adresse complete
5. Partagez votre position GPS (ou passez cette etape)
6. Envoyez des photos de votre propriete
7. Tapez **"termine"** quand vous avez fini
8. Confirmez les informations

## Enregistrer un client

### Obligation legale

Tout hebergeur doit declarer ses clients dans les **24 heures** suivant leur arrivee. La non-declaration peut entrainer des sanctions.

### Via le portail web

1. Allez dans **"Clients"** > **"Nouvel arrivee"**
2. Selectionnez la propriete concernee
3. Remplissez les informations du client:

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| Prenom | Prenom du client | Oui |
| Nom | Nom de famille | Oui |
| Date de naissance | Pour determiner l'age | Oui |
| Nationalite | Pays d'origine | Oui |
| Type de document | CNI, Passeport, Carte CEDEAO | Oui |
| Numero de document | Numero sur le document | Oui |
| Date d'expiration | Date de validite | Non |
| Telephone | Numero de contact | Non |
| Email | Adresse email | Non |

4. Telechargez une photo du document d'identite
5. Indiquez le nombre de nuits prevues
6. Cliquez sur **"Enregistrer l'arrivee"**

### Via WhatsApp

1. Selectionnez **"Nouveau locataire"** dans le menu
2. Envoyez une photo du passeport ou de la CNI du client
3. Le systeme analysera automatiquement le document
4. Verifiez les informations detectees et confirmez
5. Indiquez le nombre de nuits

### Cas special: Client mineur

Si le client a moins de 18 ans:

1. Le systeme vous demandera si un accompagnateur est present
2. Si oui:
   - Entrez le nom de l'accompagnateur
   - Indiquez le lien avec le mineur (parent, tuteur, etc.)
   - Fournissez le document d'identite de l'accompagnateur
3. Si non:
   - **Attention**: Un mineur non accompagne declenche une alerte aux autorites

## Depart d'un client

### Via le portail web

1. Allez dans **"Clients"** > **"Clients actuels"**
2. Trouvez le client dans la liste
3. Cliquez sur **"Enregistrer le depart"**
4. Le systeme calculera automatiquement:
   - Nombre de nuits
   - Montant de la TPT due
5. Confirmez le depart

### Via WhatsApp

1. Selectionnez **"Depart locataire"** dans le menu
2. Selectionnez le client dans la liste
3. Confirmez le depart

## Payer la TPT

### Calcul de la TPT

La Taxe de Promotion Touristique est calculee comme suit:

```
TPT = 1,000 FCFA x Nombre de nuits x Nombre de personnes
```

**Exemples:**
- 1 personne, 3 nuits = 3,000 FCFA
- 2 personnes, 5 nuits = 10,000 FCFA

### Modes de paiement

| Mode | Disponibilite |
|------|---------------|
| Wave | Oui |
| Orange Money | Oui |
| Carte bancaire | Bientot disponible |

### Via le portail web

1. Allez dans **"Paiements"** > **"Payer la TPT"**
2. Vous verrez le montant total du:
   - Nombre de declarations non reglees
   - Montant total a payer
3. Cliquez sur **"Payer"**
4. Choisissez votre mode de paiement:
   - **Wave**: Vous recevrez une demande de paiement sur votre application Wave
   - **Orange Money**: Composez #144# ou utilisez l'application Orange Money
5. Validez le paiement avec votre code secret
6. Vous recevrez un recu par SMS et email

### Via WhatsApp

1. Selectionnez **"Payer TPT"** dans le menu
2. Le systeme affichera votre solde
3. Choisissez **Wave** ou **Orange Money**
4. Confirmez le paiement
5. Suivez les instructions pour valider sur votre application de paiement
6. Vous recevrez la confirmation par WhatsApp

### Telecharger un recu

1. Allez dans **"Paiements"** > **"Historique"**
2. Cliquez sur le paiement souhaite
3. Cliquez sur **"Telecharger le recu"**

## Consulter l'historique

### Historique des clients

1. Allez dans **"Clients"** > **"Historique"**
2. Utilisez les filtres:
   - Par propriete
   - Par periode
   - Par nationalite
3. Exportez en CSV si necessaire

### Historique des paiements

1. Allez dans **"Paiements"** > **"Historique"**
2. Consultez tous vos paiements
3. Telechargez les recus

### Statistiques

Consultez vos statistiques:
- Nombre de clients par mois
- Taux d'occupation
- TPT payee vs due
- Repartition par nationalite

## Questions frequentes

### Inscription et compte

**Q: J'ai perdu mon telephone, comment acceder a mon compte?**

R: Contactez le support au +221 XX XXX XX XX avec votre numero de CNI pour recuperer l'acces.

**Q: Puis-je avoir plusieurs proprietes sur un meme compte?**

R: Oui, vous pouvez enregistrer autant de proprietes que vous le souhaitez.

**Q: Combien de temps prend la validation d'une propriete?**

R: La validation prend generalement 24 a 48 heures ouvrees.

### Enregistrement des clients

**Q: Que faire si le document d'identite du client est illisible?**

R: Vous pouvez saisir manuellement les informations et joindre une photo du document.

**Q: Dois-je enregistrer les clients de nationalite senegalaise?**

R: Oui, tous les clients doivent etre enregistres, quelle que soit leur nationalite.

**Q: Que se passe-t-il si j'oublie d'enregistrer un client?**

R: Vous risquez une amende. Nous vous envoyons des rappels pour eviter les oublis.

### TPT et paiements

**Q: Quand dois-je payer la TPT?**

R: La TPT est due dans les 30 jours suivant le depart du client.

**Q: Puis-je payer en plusieurs fois?**

R: Oui, vous pouvez payer les declarations individuellement.

**Q: Comment obtenir une facture pour ma comptabilite?**

R: Chaque paiement genere un recu officiel telechargeable depuis votre espace.

**Q: Que faire si mon paiement Wave/Orange Money echoue?**

R: Attendez quelques minutes et reessayez. Si le probleme persiste, contactez le support.

### Problemes techniques

**Q: L'application ne reconnait pas le document d'identite**

R: Assurez-vous que la photo est nette et bien eclairee. Vous pouvez toujours saisir les informations manuellement.

**Q: Je n'ai pas recu le code SMS**

R: Verifiez que votre numero est correct et attendez 2 minutes. Cliquez sur "Renvoyer le code" si necessaire.
